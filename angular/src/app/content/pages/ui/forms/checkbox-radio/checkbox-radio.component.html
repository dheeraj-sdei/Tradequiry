It is very easy. Add the class .checkbox to <label>.

Square

Add .checkbox-square to .checkbox.

Inline

Add .checkbox-inline for inline checkboxes.

Check list

Wrap th list of checkboxes and radio buttons to .check-list and margin will be added to them.

It is very easy. Add the class .radio to <label>.

Inline

Add .radio-inline for inline radio buttons.

Check list

Wrap th list of checkboxes and radio buttons to .check-list and margin will be added to them.

Add .checkbox-* for to change the appearance of a checkbox: .checkbox-primary, .checkbox-info...

In addition to the .checkbox-* modification classes you can also add an additional class .checkbox-grey.

Outline

Add .checkbox-outline-* for outline checkboxes: .checkbox-outline-primary, .checkbox-outline-info...

Add .radio-* for to change the appearance of a radio buttons: .radio-primary, .radio-info...

In addition to the .radio-* modification classes you can also add an additional class .radio-grey.

Outline

Add .radio-outline-* for outline radio buttons: .radio-outline-primary, .radio-outline-info...
